The AI assistant is online-only: it runs only when this computer has internet and a valid cloud API key (Gemini, Groq, or OpenRouter). There is no offline FAQ.

Enable the assistant

  1. Get a free key from Google AI Studio (search: Google AI Studio API key).
  2. In your project folder (next to backend), create .env with: GOOGLE_API_KEY=paste_key_here
  3. Run pip install python-dotenv if needed.
  4. Restart Flask (python app.py from the backend folder).
  5. Refresh this page. The chat will appear when the key loads.

Optional keys: GROQ_API_KEY, OPENROUTER_API_KEY. See Settings.
