Setup checklist

  1. Get a free API key: Google AI Studio (Gemini) · Groq · OpenRouter
  2. Create or edit this file on your machine (same folder as backend):
    {{ env_file_path }}
  3. Add a line: GOOGLE_API_KEY=your_key_here
  4. Install deps if needed: pip install python-dotenv google-generativeai
  5. Restart Flask from the backend folder, then refresh Settings.

Order of use: Gemini → Groq → OpenRouter. Needs internet on the computer running the app.
